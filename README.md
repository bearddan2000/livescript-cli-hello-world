# livescript-cli-hello-world

## Description
A POC for livescript programming language.

## Tech stack
- javascript
- nodejs
    - npm

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
